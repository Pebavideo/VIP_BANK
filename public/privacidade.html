Política de Privacidade

Proteção de Seus Dados

No VIP BANK, levamos a privacidade e a segurança dos seus dados muito a sério. Esta política descreve como coletamos, usamos e protegemos suas informações em nossa plataforma de simulação bancária.

Importante: VIP BANK é um simulador bancário educacional. Todas as operações são fictícias e não representam transações financeiras reais.

Dados Coletados

Coletamos apenas as informações essenciais para proporcionar uma experiência de simulação bancária realista e personalizada:

Informações Pessoais

  • Nome Completo: Para personalização da interface do usuário
  • CPF: Para simulação de identificação bancária (formato válido apenas)
  • E-mail: Para identificação única da conta simulada
  • Celular: Para simulação de contato e notificações
  • Senha Transacional: Para autenticação de operações simuladas

Dados de Uso

  • Saldo e Transações: Histórico completo de operações simuladas
  • Preferências de Interface: Configurações personalizadas do usuário
  • Data e Hora: Registro temporal das atividades

Uso dos Dados

Seus dados são utilizados exclusivamente para:

  • Simulação de Transações: Processar transferências PIX, depósitos e pagamentos fictícios
  • Personalização: Adaptar a experiência do usuário com base no perfil criado
  • Autenticação: Validar operações através da senha transacional
  • Histórico: Manter registro completo das atividades simuladas
  • Demonstração: Exibir capacidades tecnológicas da plataforma

Nunca compartilhamos seus dados com terceiros. Todas as informações permanecem no seu dispositivo.

Segurança e Armazenamento

Armazenamento Local

Todos os seus dados são armazenados exclusivamente no seu dispositivo através do localStorage do navegador. Isso significa:

  • Nenhum dado é enviado para servidores externos
  • Você tem controle total sobre suas informações
  • Os dados são perdidos apenas se você limpar o cache do navegador
  • Não temos acesso às suas informações após o armazenamento

Criptografia e Proteção

  • Senha Transacional: Armazenada de forma segura no localStorage
  • Dados Sensíveis: Protegidos pelas barreiras de segurança do navegador
  • Acesso Local: Apenas seu dispositivo pode acessar os dados

Responsabilidade do Usuário

Como seus dados ficam no seu dispositivo, recomendamos:

  • Manter seu dispositivo seguro com senha/biometria
  • Não compartilhar sua senha transacional
  • Utilizar senhas numéricas de 6 dígitos não óbvias
  • Limpar dados do navegador ao trocar de dispositivo

LGPD - Lei Geral de Proteção de Dados

Estamos em total conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018). Seus direitos como titular de dados:

Seus Direitos

  • Acesso: Visualizar todos os dados armazenados no seu dispositivo
  • Correção: Atualizar informações pessoais a qualquer momento
  • Exclusão: Remover todos os dados limpando o cache do navegador
  • Portabilidade: Exportar seus dados através das ferramentas de desenvolvedor
  • Informação: Acesso completo a esta política a qualquer momento

Como Exercer Seus Direitos

Como os dados ficam no seu dispositivo, você pode:

  • Acessar as configurações do seu navegador
  • Limpar dados de navegação e cache
  • Usar as ferramentas de desenvolvedor para inspecionar o localStorage
  • Entrar em contato através do suporte para orientações técnicas

Menores de Idade

O VIP BANK é destinado a maiores de 18 anos ou menores com autorização parental. Recomendamos:

  • Supervisão parental para menores de 18 anos
  • Explicação sobre natureza simulada da plataforma
  • Orientação sobre segurança de senhas e dados pessoais
  • Limitação de tempo de uso para menores

Atualizações desta Política

Podemos atualizar esta política ocasionalmente para:

  • Refletir mudanças em nossas práticas de dados
  • Atender a novas exigências legais
  • Melhorar a transparência com nossos usuários
  • Adaptar a novas funcionalidades da plataforma

Qualquer alteração significativa será comunicada através de aviso na aplicação. A data da última atualização sempre estará disponível no topo deste documento.

Contato e Suporte

Para dúvidas sobre esta política ou exercício de direitos LGPD:

  • Natureza: Somos uma plataforma educacional de demonstração
  • Disponibilidade: Suporte técnico através da documentação
  • Respostas: Orientações sobre gerenciamento de dados locais
  • Tempo: Respostas em até 7 dias úteis
